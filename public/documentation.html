Documentation: Movie API

Request URL Response
GET http://localhost:8080/movies A list of all movies
GET http://localhost:8080/movies/:title Data on a single movie
GET http://localhost:8080/genre/:name Data about a genre by name/title
GET http://localhost:8080/directors/:name Data about a director by name
PUT http://localhost:8080/user/newuser Register new user
POST http://localhost:8080/user/:name update username
PUT http://localhost:8080/user/:name/favoritelist add a movie
DELETE http://localhost:8080/user/:name/favoritelist/:moviename delete a movie
DELETE http://localhost:8080/user/:name delete a user

/movies

Get request

Returns a json file with data on all movies

No data in request body necessary

Response body format: JSON

Query parameters: movies


/movies/:title

Get request

Returns a json file with data on a specific movie

No data in request body necessary

Response body format: JSON

Query parameters: movies, title


/genre/:name

Get request

Returns a json file with data on a specific genre

No data in request body necessary

Response body format: JSON

Query parameters: genres, name


/directors/:name

Get request

Returns a json file with data on a specific director

No data in request body necessary

Response body format: JSON

Query parameters: directors, name


/user/newuser

PUT request

Sends a message that a new user has registered sucessfully

JSON file with user data

Response body format: plain text

Query parameters: user, newuser


/user/:name

POST request

Sends a message that a has updated his/her name

JSON file with old name und new name

Response body format: plain text

Query parameters: user, name


/user/:name/favoritelist

PUT request

Sends a message that new movie has been added to favoritelist

JSON file with movie data

Response body format: plain text

Query parameters: user, name, favoritelist


/user/:name/favoritelist/:moviename

DELETE request

Sends a message that new movie has been deleted from the favoritelist

no request body necessary

Response body format: plain text

Query parameters: user, name, favoritelist, moviename


/user/:name

DELETE request

Sends a message that the user has successfully deregestired

no request body necessary

Response body format: plain text

Query parameters: user, name
