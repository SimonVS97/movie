Documentation: Movie API

Request URL Response
GET https://movie-app-svs.herokuapp.com/movies This request does not need a body. It returns a list of all movies in a JSON format.
GET https://movie-app-svs.herokuapp.com/movies/:title This request requires the title of a movie as a parameter. It returns information on the movie in a JSON format.
GET https://movie-app-svs.herokuapp.com/genre/:name This request requires the name of the genre as a parameter. It returns the information on the genre in a JSON format.
GET https://movie-app-svs.herokuapp.com/directors/:name This request requires the name of the director as a parameter. It returns the information on the director in a JSON format./td>
GET https://movie-app-svs.herokuapp.com/users This request does not need a body. It returns information on all users.
GET https://movie-app-svs.herokuapp.com/users/:Username This request requires the name of a user as a parameter. It returns the information on that user in a JSON format.
POST https://movie-app-svs.herokuapp.com/users This request requires a body that contains the information for the new user in a JSON format.
It creates a new user in the database. It returns the information of the newly created user in a JSON format.
PUT https://movie-app-svs.herokuapp.com/user/:Username This request requires a body that contains the information that is supposed to replace the existing information of the user.
It updates the information of the user in the database. It returns the updated information of the user in a JSON format.
POST https://movie-app-svs.herokuapp.com/users/:Username/movies/:MovieID This request requires a body that contains information on the specific user. In addition, the movie is required as a parameter.
It adda a movie to the favorite list of the user. It will return the data of the user in a JSON format.
DELETE https://movie-app-svs.herokuapp.com/users/:Username/movies/:MovieID This request requires the name of the user and the movie to be deleted from his favorite list as parameters.
It deletes a movie from the favorite list. It reutrns the updated information of the user in a JSON format.
DELETE https://movie-app-svs.herokuapp.com/users/:Username This request requires the name of the user as a parameter. It deletes the user from the database. It returns a message.
